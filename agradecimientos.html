AGRADECIMIENTOS.

Puedes pulsar en el logotipo para acceder a su página web

TemplateFlip:

1º. Para hacer este Currículo Vitae he usado una plantilla de TemplateFilp, que pueden ver pulsando aquí

Bootstrap:

2º. Cómo esta plantilla estaba realizada en Bootstrap, de la página web oficial he podido resolver las dudas. Si quieren aprender a usar esta herramienta, pueden ir directamente a aquí

Wikimedia Commons:

3º. De Wikimedia Commons saqué la foto de Arucas de la cabecera. Es una web que fue creada para servir como almacén de archivos, y donde pueden descarlo directamente, usarlo en vuestra web, entre otras funciones. Si quieren ver la foto directamente y las funciones que esta web ofrece, pueden hacerlo pulsando aquí

Zimrre:

4º. Zimrre.com es el mejor generador de textos legales para una web que he visto, y además es gratuito. Eso sí, en el caso de quererlo para tu página web, luego tendrás que convertirlo en código. Les dejo como ejemplo el generador de la política de privacidad, pueden verlo pulsando aquí

Peengler:

5º. Peengler es una web donde encontrarás muchísimos tutoriales que podrás aplicar a tu página. Gabriela es la que graba los tutoriales, y de ella y su página también he podido ver como se hacen los documentos legales necesarios, que en este caso, el que he escogigo ha sido el de cookies, puesto que el resto me gustó mas la web anteriormente mencionada (aunque en el caso de ambas web recomiento leer bien lo que dice, y modificar lo que sea necesario, o adaptarlo a tu página), y también he aprendido muchas otras cosas, como por ejemplo los botones. Como muestra les dejo su página donde habla de los textos legales, por si quieren comparar con la anterior web. Pueden verlo pulsando aquí

FalconMasters:

6º. Carlos Arturo es quien hace los tutoriales para YouTube de FalconMasters, entre esos tutoriales, el que he utilizado para realizar esta web, es precisamente el del aviso de cookies obligatorio, que debe aparecer como pantalla emergente la primera vez que accedes a la misma. Este vídeo pueden verlo pulsando aquí

Font Awesome:

7º. Una de las webs que mas he usado para obtener los iconos es Font Awesome. Entre sus ventajas está el que puedes cambiar el color, poner efectos prediterminados, así como elegir alguno de los tamaños predefinidos que tiene, y coger directamente el código html de su web. Si quieren ver los iconos de los que dispone esta web, pueden verlos pulsando aquí

Flaticon:

8º. Flaticon es otra web que hemos usado para los iconos, sin embargo aquí aunque tienes muchos modelos para elegir, el tamaño viene predefinido. Si quieres ver los ejemplos que tienen para el icono del WhatsApp, pueden verlos pulsando aquí

Si quieres ver un del icono del WhatsApp, para que de esa forma te sea más fácil la comparación, puedes verlo pulsando aquí

EMOJI CSS:

10º. En este caso va a coincidir el siguiente enlace con el del título, puesto que solo es una página web sencilla, pueden verlos pulsando aquí
